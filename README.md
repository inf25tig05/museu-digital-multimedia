# Museu Digital Multimédia

Repositório criado para alojar o projeto desenvolvido no âmbito da unidade curricular de multimédia. 

Projeto desenvolvido pelo grupo inf25tig05. 

## Descrição do projeto

O Museu Digital Multimédia é um website interativo que simula a experiência de visita a um museu digital. O projeto apresenta uma entrada visual e três salas principais: 

- Sala de imagem
- Sala de som
- Sala de vídeo

O objetivo foi criar uma experiência simples, visual e interativa, utilizando conteúdos multimédia como imagens, áudio e vídeo.

## Organização do repositório

O projeto está organizado da seguinte forma:

- `index.html` contém a página inicial do museu.
- `sala-imagem.html` contém a sala dedicada às obras visuais.
- `sala-som.html` contém a sala dedicada aos conteúdos sonoros.
- `sala-video.html` contém a sala dedicada ao vídeo.
- A pasta `css` contém os ficheiros de estilo.
- A pasta `js` contém os ficheiros JavaScript.
- A pasta `img` contém as imagens utilizadas no projeto.
- A pasta `audio` contém os ficheiros de som.
- A pasta `video` contém o vídeo utilizado no site.
- A pasta `xml` contém os ficheiros XML e XSD.
- A pasta `doc` contém os capítulos do relatório em Markdown.


## Tecnologias utilizadas

Neste projeto utilizamos as seguintes tecnologias: 

- HTML5
- CSS3
- JavaScript
- XML
- XSD
- Markdown

## Frameworks e bibliotecas Não foram utilizadas frameworks externas.

O projeto foi desenvolvido com HTML, CSS e JavaScript puro. 

## Responsividade

O website foi desenvolvido para dois tipos de ecrã: 

- Desktop / notebook
- Tablet

A versão desktop foi criada como versão principal. A adaptação para tablet foi feita através de CSS, utilizando media queries.
