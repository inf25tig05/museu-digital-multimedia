# Capítulo 1: Apresentação do projeto

## Introdução

A proposta surgiu da ideia de transformar a navegação tradicional de um site numa experiência mais visual e imersiva. Em vez de apresentar apenas páginas estáticas, o projeto conduz o utilizador por uma entrada cinematográfica até diferentes salas temáticas.

Cada sala foi pensada para explorar um tipo de conteúdo multimédia, criando uma ligação entre imagem, som e vídeo dentro do mesmo ambiente virtual. Desta forma, o museu funciona como uma experiência digital simples, organizada e interativa.

## Tema do projeto

O tema escolhido foi um **museu digital multimédia**.

A escolha deste tema permitiu reunir diferentes tipos de conteúdo num único website, criando uma experiência mais completa para o utilizador. A ideia foi construir um ambiente visual inspirado num museu, mas adaptado ao formato digital e interativo.

## Objetivos

Os principais objetivos do projeto foram:

- desenvolver um website multimédia;
- criar uma página inicial com uma experiência visual de entrada no museu;
- criar salas temáticas para imagem, som e vídeo;
- utilizar HTML para estruturar as páginas;
- utilizar CSS para criar o design visual do projeto;
- utilizar JavaScript para adicionar interações;
- utilizar XML para organizar informação de forma estruturada;
- utilizar conteúdos multimédia, como imagens, áudio e vídeo;
- adaptar o website para dois tipos de ecrã;
- organizar o projeto num repositório GitHub.

## Público-alvo

O público-alvo do projeto são utilizadores interessados numa experiência digital simples, visual e interativa.

O website foi pensado para ser fácil de utilizar, sem exigir conhecimentos técnicos do utilizador. A navegação é feita através de elementos visuais e botões, permitindo aceder às diferentes salas do museu.

## Descrição geral do website

O website começa com uma página inicial que funciona como entrada do museu digital. Esta página apresenta uma sequência visual que conduz o utilizador até ao interior do museu.

Depois da entrada, o utilizador tem acesso às diferentes salas do projeto:

### Sala de imagem

A sala de imagem apresenta obras visuais e informações associadas.  
Os dados desta sala são organizados com recurso XML, permitindo separar a informação do conteúdo visual da estrutura HTML.

### Sala de som

A sala de som apresenta conteúdos sonoros.  
Nesta sala, o utilizador pode interagir com elementos relacionados com música e ouvir ficheiros de áudio integrados no projeto.

### Sala de vídeo

A sala de vídeo apresenta um conteúdo audiovisual.  
Esta página foi criada para simular uma sala de projeção dentro do museu digital.

## Conclusão

A proposta do museu digital foi escolhida porque permitia trabalhar diferentes conteúdos multimédia dentro de uma mesma experiência. A página inicial, as salas e os elementos interativos foram pensados para criar a sensação de entrada num espaço virtual, e não apenas de navegação entre páginas comuns.

Durante o processo, algumas partes exigiram mais ajustes, principalmente a organização visual, o funcionamento do scroll no javascript e o posicionamento dos elemento. Essas etapas foram importantes para melhorar o funcionamento do site e perceber melhor como pequenas alterações no código influenciam a experiência do utilizador.
